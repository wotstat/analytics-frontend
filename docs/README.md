## Ключевые факты (TL;DR)

- SPA на **Vue 3** (Composition API, `<script setup>`) + **TypeScript** + **Vite**, пакетный менеджер — **Bun**.
- Сайт **wotstat.info** — аналитика по игре «Мир танков» / World of Tanks на основе событий, собираемых игровым модом.
- Данные фронтенд берёт **напрямую из ClickHouse** (через web-прокси, пользователь `public`, база `WOT`) — бэкенда с REST API почти нет.
- Языки интерфейса: в основном русский (тексты захардкожены по-русски); точечная i18n есть только в отдельных местах.
- Тестов в проекте **нет**. Проверка — `bun run build` (включает `vue-tsc`) и ESLint.
- Стейт-менеджера (Pinia/Vuex) нет: состояние — локальное, `useLocalStorage` из VueUse и URL query-параметры.
