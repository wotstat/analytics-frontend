# [wotstat.info](https://wotstat.info) — аналитика для «Мира танков» и World of Tanks

Это исходный код сайта **wotstat.info**.

WotStat — бесплатный сервис аналитики для игр «Мир танков» (Lesta) и World of Tanks (Wargaming). Игроки устанавливают мод, который во время боёв собирает игровые события (выстрелы, урон, результаты боёв и многое другое) и отправляет их в общую базу данных. Сайт превращает эти данные в наглядную статистику:

- 📊 **Сессионная инфографика** — личная статистика: винрейт, точность стрельбы, урон, время в очереди, статистика по картам и танкам;
- ⚔️ **Натиск** — подробная статистика рейтингового режима и таблица лидеров, обновляющаяся в реальном времени;
- 🎯 **Распределение урона** — калькулятор реального разброса разового урона орудия;
- 📼 **Реплеи** — поиск и анализ реплеев;
- 🧩 **Виджеты** — оверлеи для стримов (OBS) и игры;
- 🔧 **Установщик модов** — установка модов в один клик прямо из браузера;
- 🎉 Инфографика ивентов: лутбоксы, «Битва Блогеров» и другие.

Все собранные данные открыты: любой желающий может получить прямой SQL-доступ к базе и строить собственную аналитику.

## Для разработчиков

SPA на **Vue 3** (Composition API + TypeScript) и **Vite**, пакетный менеджер — **Bun**. Данные запрашиваются напрямую из ClickHouse через web-клиент.

```bash
bun install      # установить зависимости
bun run dev      # dev-сервер (vite --host)
bun run build    # проверка типов (vue-tsc) + прод-сборка
bun run preview  # предпросмотр сборки
```

Подробная документация по архитектуре и кодовой базе — в [docs/](docs/README.md).
